Maze Solving Car

Maze Solving Car

Finished in December, 2019
Image

A picture of the finished product

The car has sensors on the left, right and front which can detect for walls and can determine a map of the maze. The car first traverses through the maze and determines the structure of the maze, then the car with travel through the maze again using the shortest path possible by using an algorithm. The motors have encoders of them which enables for more precise tracking of the cars movement which allows for more precision when turning and moving.


Tower Defense Game

Finished in January, 2019
Image

A screenshot of the game

A tower defense game that was built based on a 2D array. The grid based system allows for automatic path generation based on user inputs. The game was built from strach by myself and four friends. The game includes music, settings, save files, achievements and many other features. The game has three unique stages and an endless round which has randomly generated enemies which increases in difficulty as the user progresses.

Vex Robotic Competition

Finished in January, 2019
Image

A picture of the finished product

Vex Robotic Competition is a Canadian robitcs competition where every year they propose a new challenge. The 2019 challenge was to score as many points as possible. Some methods to score were to; retrieve balls on the field and shoot at targets, flip caps and place them on poles, and to climb a platform. Our team built a robot that was able to retrieve balls and shoot them at the targets and climb the platform. Additionally, our robot was able to travel in any direction through the use of omni wheels.
